About Us

Our Mission

Vassar in Tech's mission is to connect and empower the Vassar community at every stage of their careers in technology by fostering programming, networking, and access to opportunities that drive meaningful professional growth.

We host quarterly in-person events in New York City, along with virtual programming throughout the year.

Vassar in Tech is a volunteer-led initiative by Alison Albeck Lindland ('00), Peter Leonard ('97), and Dan Leslie ('02).

Clone This Website

Everything you need to host an exact copy of this site yourself.

1

Tech Stack

This is a plain static website — no framework, no build step, no dependencies. Just three files:

  • index.html — all page content and structure
  • styles.css — all styling and layout
  • script.js — mobile nav toggle and scroll behavior
  • logo.png — the group logo

It can be hosted anywhere that serves static files: Vercel, Netlify, GitHub Pages, etc.

2

Get the Source Code

The full source is publicly available on GitHub. Fork or clone it to your own account:

github.com/johnmhsu11/vassar-in-tech →
Clone via terminal git clone https://github.com/johnmhsu11/vassar-in-tech.git

Or click Fork on GitHub to create your own copy under your account.

3

Deploy to Vercel (Recommended)

  1. Create a free account at vercel.com
  2. Click Add New Project and import your forked GitHub repo
  3. Leave all settings as default — Vercel auto-detects static sites
  4. Click Deploy

Your site will be live at your-project-name.vercel.app in under a minute. Future pushes to GitHub redeploy automatically.

4

Customize the Content

All content lives in index.html. Key things to update:

  • Mission statement and leadership names in the About section
  • Event details in the Programming section
  • Contact email (Alison.Lindland@gmail.com)
  • LinkedIn group URL (linkedin.com/groups/3732591)
  • Colors in styles.css under :root — change --maroon and --gray
  • Replace logo.png with your own logo (same filename)
